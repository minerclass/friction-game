# Friction Lab

An interactive pedagogical-friction calibration simulation by Micah J. Miner.

The 5-minute experience asks participants to design one assignment across noetic, rhetorical, and existential friction, then secure the infrastructural conditions that allow those designs to persist.

Its central mechanic is a two-axis calibration map: productive challenge and exclusionary barriers are different dimensions, not opposite ends of one dial.

## Running Locally

Open `index.html` in a modern browser. Sound is optional and off by default.
